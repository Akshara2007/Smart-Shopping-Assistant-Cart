# Smart-Shopping-Assistant-Cart

🛒 A smart embedded system project built using Arduino that automates shopping assistance by identifying products, displaying information, and calculating total costs using RFID, sensors, and real-time processing.

## ✨ Features
-  Product identification using RFID
-  Displays product details and price
-  Automatically calculates total cost
-  Real-time shopping assistance

## 🛠️ Technologies Used
-  Arduino Uno
-  Embedded C/C++
-  RFID Module
- 📟LCD Display

## ⚙️ Working Principle
The RFID module scans product tags and sends product information to the Arduino. The Arduino processes the data, displays product details on the LCD, and updates the total cost automatically.
